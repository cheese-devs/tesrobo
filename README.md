# 🤖 Robo Rescue with microROS

โปรเจกต์หุ่นยนต์สำหรับการแข่งขัน **Robo Rescue with microROS**  
การแข่งขันหุ่นยนต์ ส.ส.ท. – สพฐ. ยุวชน ประจำปี 2569

---

## 📌 เกี่ยวกับโครงงาน
โครงงานนี้เป็นการพัฒนาหุ่นยนต์กู้ภัยที่สามารถทำงานอัตโนมัติ  
โดยใช้ **microROS** สำหรับการสื่อสารและควบคุมหุ่นยนต์

ความสามารถหลัก:
- เคลื่อนที่ตามสนามแข่งขัน
- ตรวจจับเส้นทางและสิ่งกีดขวาง
- ทำภารกิจกู้ภัยตามกติกาการแข่งขัน

---

## 🛠 เทคโนโลยีที่ใช้
- microROS
- ROS 2
- Arduino / ESP32
- ภาษา C / C++
- Sensor และ Motor Driver
